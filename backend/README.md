Doctor Appointment Booking System (Backend)

A full-stack SaaS backend system for booking doctor appointments with role-based authentication and secure APIs.
//
🚀 Features
🔐 JWT Authentication (Patient, Doctor, Admin)
👨‍⚕️ Doctor Profile Management
📅 Appointment Booking System
❌ Prevent Double Booking
🛡️ Role-Based Access Control
📊 RESTful API Architecture
//
🧱 Tech Stack
Node.js
Express.js
MongoDB (Mongoose)
JWT Authentication
//
📦 API Modules
1-Auth
  Register
  Login
2-Doctors
  Create profile
  Approve doctor
  Get doctors
3-Appointments
  Book appointment
  Cancel appointment
  View appointments
